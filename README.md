# Stack
- TS
- React
- react-router-dom
- ESLint + Prettier
- Absolute imports
- Opinionated folder structure

# Folder structure
- `src`
  - `components` - reusable components
  - `pages` - pages
  - `services` - services (axios, etc.)
  - `store` - store
  - `lib` - lib (utils, etc.)
